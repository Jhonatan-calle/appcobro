# Loan Portfolio App

Welcome to the Loan Portfolio App! This application allows you to manage loans, create customer profiles, record loan payments, and keep a history of paid-off loans. Developed using React Native for the frontend and Express along with MongoDB for the backend.

## Features

- Creation and management of customer profiles.
- Recording loans and loan payment transactions.
- Handling a history of paid-off loans.
- Quick customer search by name or ID number.
- User-friendly and easy-to-navigate interface.

## Installation and Usage

1. Clone this repository to your local machine.
2. Ensure you have Node.js and npm installed.
3. Navigate to the client folder and run `npm install` to install frontend dependencies.
4. Navigate to the server folder and run `npm install` to install backend dependencies.
5. Configure the MongoDB database connection in the backend configuration file.
6. On Windows, run the `start-app.bat` file. On other operating systems, navigate to the server folder and run `npm start dev`, then navigate to the client folder and run `npm expo start`.
